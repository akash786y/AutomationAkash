# Bike Cards Validation Test

This project contains automated tests for verifying bike listings on a web application. The tests ensure that all displayed bike cards match the expected bike model and are available for booking.

## Prerequisites

- Java 11 or higher
- Maven or Gradle
- Playwright for Java
- TestNG (or your preferred test framework)

## Installation

 Clone the repository:
 
   git clone <repository-url>
   cd <repository-folder>
   mvn clean install



###  Run the TestRunner file in src.test.java.runner Package


### Test Cases -> src.test.java.steps.RoyalBrothersSteps
